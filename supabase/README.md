# Supabase

Ampex Mobile deler PostgreSQL-skjema med Ampex Web.

- **Ikke** opprett duplikat-migrasjoner her uten avtale med web-team.
- Bruk Supabase Dashboard eller felles migrasjonsrepo for skjemaendringer.
- Mobil bruker `company_id` som tenant-nøkkel.

Dev-prosjekt konfigureres via `.env` i app-roten.
